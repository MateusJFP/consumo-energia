⚡ Calculadora de Consumo de Energia

📌 Sobre o projeto
Este projeto é uma calculadora de consumo de energia elétrica feita em Python.
O objetivo é ajudar o usuário a estimar quanto um aparelho consome por mês e o custo aproximado desse consumo.

🧠 Como funciona
O programa solicita:
    • Nome do aparelho
    • Potência (em watts)
    • Tempo de uso diário (em horas)
E calcula o consumo mensal com a fórmula:
consumoMensal = (potencia * horasDia * 30) / 1000
Também calcula o custo estimado com base em um valor fixo de kWh.

💻 Tecnologias utilizadas
    • 🐍 Python
    • 🧮 Lógica de programação
    • 📂 Git e GitHub

▶️ Como executar
    1. Clone o repositório:
git clone https://github.com/MateusJFP/consumo-energia.git
    2. Acesse a pasta:
cd consumo-energia
    3. Execute o programa:
python app.py

📊 Exemplo de saída
Aparelho: Geladeira
Consumo estimado: 45.00 kWh/mês
Custo estimado: R$ 33.75
